### How to start

1. Clone this repo
2. cd web-ui
3. npm install
4. npm run dev
5. cd ..
6. cd indexing-service
7. python -m venv .venv
8. source .venv/bin/activate
9. pip install -r requirements.txt
10. cd ..
11. docker-compose up
